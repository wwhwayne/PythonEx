# 默认快捷键 
#### 来自 https://resources.jetbrains.com/storage/products/intellij-idea/docs/IntelliJIDEA_ReferenceCard.pdf

# 记住这些快捷键
* 【Ctrl + Shift + Space】        智能代码补全
* 【Double Shift】                全局搜索 
* 【Alt + Enter】                 显示推荐操作及快速修复 
* 【Alt + Ins】                   生成代码 
* 【Ctrl + P】                    参数信息 
* 【Ctrl + W】                    扩展选择 
* 【Ctrl + Shift + W】            缩小选择 
* 【Ctrl + E】                    最近文件弹窗 
* 【Shift + F6】                  重命名
